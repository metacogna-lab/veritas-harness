# Guides

These are operator guides for the Veritas meta-harness. Read them in order. Each part assumes the previous one is done.

| Part | File | What you learn |
|------|------|----------------|
| 1 | [01-create.md](01-create.md) | How to create a harness under `harness/` |
| 2 | [02-init.md](02-init.md) | How to install and verify a harness |
| 3 | [03-claude-code.md](03-claude-code.md) | How to run missions with Claude Code as the LLM |
| 4 | [04-ollama-hermes.md](04-ollama-hermes.md) | How to run missions with Hermes3 on Ollama |

After the four parts, work through [EXAMPLE.md](EXAMPLE.md). That file is the canonical end-to-end walkthrough. An older example lives at `harness/veritas-research/EXAMPLE.md`; use this folder instead.

Reference docs (not duplicated here): [harness/INSTALLATION.md](../harness/INSTALLATION.md), [harness/veritas-research/PROVIDER.md](../harness/veritas-research/PROVIDER.md).
