# AGENDA — Solo Sitting

1. **Open** — loadout + kit index (5 min)
2. **Frame** — review PROBLEM / CONSTRAINTS / SUCCESS (5–20 min)
3. **Cut** — lock MVP (15 min)
4. **Build** — checklist-driven slice (remainder)
5. **Demo** — fill DEMO paths, run smoke/tests (30 min)
6. **Close** — decisions log, stop on timer

Facilitator (loadout) owns the clock. Builder owns the slice. Scout only when scope is unclear.
